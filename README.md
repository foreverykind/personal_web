# Personal Portfolio

Responsive portfolio website showcasing product management and delivery expertise.

## Project Structure

```
personal_web/
├── index.html          # Main portfolio page
├── .gitignore          # Git ignore rules
├── README.md           # Project documentation
└── assets/            # (future) Images, icons, etc.
```

## Features

- Modern dark theme with custom CSS
- Fully responsive design (mobile, tablet, desktop)
- Smooth animations and interactions
- Work case studies with expandable details
- AI workflows section
- Side projects showcase
- Contact section with social links

## How to Use

1. Clone this repository
2. Open `index.html` in your browser
3. Make changes and commit with Git
